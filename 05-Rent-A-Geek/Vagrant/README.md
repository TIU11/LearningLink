# Vagrant

## Download VirtualBox

[VirtualBox](https://www.virtualbox.org/)

## Download Vagrant

[Vagrant](https://www.vagrantup.com/downloads)

## Commands

#### Initiates a VM
`vagrant init`

### Update the configuration and shell scripts

update:   [Vagrantfile](./Vagrantfile)

verify:   [kolibrisetup.sh](./kolibrisetup.sh)

#### Starts the VM
`vagrant up`

#### Stops the VM
`vagrant halt`

#### Destroys the VM
`vagrant destroy`
